# 🚗 Car Price Analysis using Python & Pandas

## 📌 Project Overview

This project analyzes a car price dataset using **Python** and the **Pandas** library. The objective is to perform data analysis by answering various business and statistical questions related to car prices, mileage, brands, fuel types, transmission types, and engine sizes.

This project helped me strengthen my knowledge of **Data Analysis**, **Pandas**, **Data Filtering**, **Grouping**, and **Aggregation** techniques.

---

## 🎯 Project Objectives

* Analyze car price data.
* Perform data filtering and aggregation.
* Calculate averages and correlations.
* Find highest and lowest values.
* Count records based on different conditions.
* Practice real-world data analysis using Pandas.
